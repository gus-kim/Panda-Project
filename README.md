# Panda-Project


- Projeto panda de análise exploratória.
- Link do dataset: [Dataset](https://www.kaggle.com/datasets/zahranusrat/diamonds-characteristics-and-pricing-analysis?select=diamonds.csv)
- Vamos fazer uma análise exploratória do dataset de diamantes,tentando formular hipóteses com base na análise, dessa forma, criando váriaveis na engenharia de variáveis para tentar capturar o comportamento do dados e comprovar nossas hipóteses formuladas.
